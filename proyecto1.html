Proyecto 1: Sistema de Recursos Humanos (RRHH)

Aplicación web empresarial desarrollada en C#, ASP.NET MVC y SQL Server

Volver al Inicio

📋 Descripción del Proyecto

El Sistema de Recursos Humanos (RRHH) es una aplicación web completa desarrollada bajo una arquitectura en capas que permite gestionar la información del personal de una empresa. El sistema incluye módulos para funcionarios, planillas, tractos de impuesto de renta, créditos fiscales, cargas sociales y horas laboradas .

Además, implementa un sistema de autenticación segura con cookies temporales y validación por roles y departamentos, asegurando que solo los usuarios autorizados puedan acceder a las funciones administrativas.

🧠 Tecnologías Utilizadas

  • 💻 Lenguaje principal: C# (ASP.NET MVC 6)
  • 🗄️ Base de datos: SQL Server 2022
  • 🎨 Interfaz: HTML, CSS, JavaScript, Bootstrap
  • ⚙️ Arquitectura: 5 Capas (UI, Negocio, Datos, DatosSQL y Entidades)
  • 🔒 Seguridad: Autenticación con Cookies + [Authorize]
  • 📈 Funcionalidades: CRUD, cálculos automáticos de planilla, validaciones y sincronización entre tablas

🧾 Detalles Técnicos

El proyecto fue desarrollado durante un mes completo de trabajo continuo, aplicando buenas prácticas de desarrollo, principios de seguridad y separación de capas. Se utilizó Visual Studio 2022 como entorno de desarrollo y SQL Server Management Studio para la administración de la base de datos.

Cada módulo está conectado por procedimientos almacenados, con control de errores y validaciones en la capa de negocio. Además, se implementó un sistema de autenticación con cookies que expiran al cerrar sesión, garantizando seguridad y trazabilidad.
