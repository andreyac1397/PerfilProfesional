Sobre el Proyecto

Javi es un chatbot enfocado en el mundo culinario, capaz de generar recetas, pasos de preparación y listas de ingredientes usando la API de OpenAI. Fue desarrollado con un diseño moderno, completamente responsive, e implementado mediante HTML, CSS y JavaScript, con despliegue en GitHub Pages. El proyecto está disponible en: andreyac1397.github.io/chatbot-javi

🧩 Tecnologías Utilizadas

  • 🌐 HTML / CSS / JavaScript
  • 🤖 OpenAI API (Chat inteligente)
  • 📦 GitHub Pages (Despliegue)
  • 🎨 Diseño moderno y responsive
